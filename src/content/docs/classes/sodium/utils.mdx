Utility helpers: constant-time comparison, memory zeroing, and hex/base64 encode/decode.
All encoding and decoding goes through libsodium, ensuring compatibility with other libsodium consumers.

### memcmp

> <LuaDeclaration name="memcmp" className="Utils" args={[ { name: "a", type: "buffer" }, { name: "b", type: "buffer" } ]} returnType="boolean" />

> <LuaDeclaration name="memzero" className="Utils" args={[ { name: "pointer", type: "FFIPointer" }, { name: "length", type: "number" } ]} />

### bin2base64

> <LuaDeclaration name="bin2base64" className="Utils" args={[ { name: "data", type: "buffer" }, { name: "variant", type: "number?" } ]} returnType="string" />
