### Utilities.tableReflect

> <LuaDeclaration name="Utilities.tableReflect" className="Utilities" args={[  ]} returnType="thread" />

Will swap the K, V's, as well as keep the original K, Vs, for example:

```luau
local a = { a = 1, b = 2, c = 3 }

tableReflect(a)

--[[
a = {
	-- default values
	a = 1,
	b = 2,
	c = 3,
	-- reflected values
	[1] = "a",
	[2] = "b",
	[3] = "c"
}
]]
```
